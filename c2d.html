From Cloud to Dirt

「從雲端到泥土」(from cloud to dirt, c2d)是一系列關於 Haskell 的教學/筆記文章。這一系列的文章試圖從高階的抽象理論開始一路往低階的 STG machine 去解釋一些名詞和觀念。最終目標是進一步說明與理解 Haskell 的運行與操作方法。追本朔源，這系列的文章始於一個基本的問題「什麼是資料型別(datatype)？」，這個問題又可以進一步分成「Haskell 中的資料型別」、「數學模型中的資料型別」、「實作細節中的資料型別」這三個微略不同的層次。因此，根據抽象的程度，這個系列的文章大略會被切割成三個主題：

  • 數學模型與指稱語意
  • Haskell 中的資料結構與計算
  • GHC 與 STG Machine 中的底層表示

此系列文章仍然是散在此 blog 中，而本篇文章則是此系列文的目錄文章。
